### Opsi 1: Jalankan sebagai Website Statis
1. Clone atau download repository ini.
2. Buka file index.html di browser web favorit Anda (Chrome, Firefox, dll.).
3. Website akan langsung berjalan tanpa server tambahan.

### Opsi 2: Menggunakan Server Lokal (Opsional)
Jika Anda ingin menjalankan dengan server lokal untuk pengembangan:
1. Pastikan Node.js terinstal di sistem Anda.
2. Jalankan perintah berikut di terminal:
   ```bash
   npm install
   npm run dev
   ```
3. Buka browser dan akses `http://localhost:3000` (atau port yang ditentukan).
